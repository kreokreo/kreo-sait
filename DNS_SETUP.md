# Настройка DNS для домена kreo.pro

## DNS записи для вашего доменного провайдера

Добавьте следующие записи в настройках DNS вашего домена:

### A запись (IPv4)
```
Тип: A
Имя: @ (или kreo.pro)
Значение: 46.149.67.149
TTL: 3600 (или Auto)
```

### AAAA запись (IPv6) - опционально
```
Тип: AAAA
Имя: @ (или kreo.pro)
Значение: 2a03:6f00:a::1:72f6
TTL: 3600 (или Auto)
```

### CNAME запись для www (опционально)
```
Тип: CNAME
Имя: www
Значение: kreo.pro
TTL: 3600 (или Auto)
```

## Пример для разных провайдеров

### Timeweb
1. Зайдите в панель управления доменом
2. Перейдите в раздел "DNS-зона" или "Управление DNS"
3. Добавьте записи:
   - **A**: `@` → `46.149.67.149`
   - **AAAA**: `@` → `2a03:6f00:a::1:72f6` (опционально)
   - **CNAME**: `www` → `kreo.pro` (опционально)

### Reg.ru
1. Зайдите в "Мои домены"
2. Выберите домен kreo.pro
3. Перейдите в "DNS-серверы и зона"
4. Добавьте записи аналогично

### Другие провайдеры
Ищите раздел:
- "DNS записи"
- "Управление DNS"
- "DNS зона"
- "DNS настройки"

## Проверка DNS записей

После добавления записей подождите 5-15 минут для распространения DNS, затем проверьте:

```bash
# Проверка A записи
dig kreo.pro A +short

# Должно вернуть: 46.149.67.149

# Проверка AAAA записи (IPv6)
dig kreo.pro AAAA +short

# Должно вернуть: 2a03:6f00:a::1:72f6
```

Или используйте онлайн-сервисы:
- https://dnschecker.org/#A/kreo.pro
- https://www.whatsmydns.net/#A/kreo.pro

## После настройки DNS

1. Подождите 5-15 минут для распространения DNS
2. Установите SSL сертификат (Let's Encrypt) на сервере
3. Настройте Nginx конфигурацию

Скрипт для установки SSL и настройки Nginx будет выполнен автоматически при следующем деплое.
